# Text-Analyzer

--> An application that analyzes the text with functionality of : <br>
&nbsp; --> Approx time to read text.<br>
&nbsp; --> counting numbers in text.<br>
&nbsp; --> counting characters in text.<br>
&nbsp; --> counting words in text.<br>
